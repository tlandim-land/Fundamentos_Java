Ambiente montado com IDE intelliJ

Atalhos do intelliJ
	psvm -> para cria o metodo main
	sout -> para criar o system.out.println();
	CTRL + D -> duplicar linha 
	CTRL + shift + f10 -> Monta ambiente de execução e executa codigo (1° vez)
	shift + f10 -> Executa codigo (Demais vez)
	alt + insert -> da opção de criar os construtores e demais metodos das classes (quando executado dentro da classe)
	ctrl + alt + f -> transforma uma variavel em um atributo da classe 
	ctrl + alt + l -> "identa" o codigo 
	ctrl + alt + o -> remove os importes não utilizados 
	ctrl + b -> quando executado sobre uma variavel, mostra aonde ela foi declarada
	ctrl + alt <  -> volta para onde o cursor estava antes do ctrl + B 
	ctrl + f4 -> fecha a classe aberta
	alt + f7 -> find usage 
	ctrl + alt + m -> após seleciona um codigo e usar esse atalho sera criado um metodo com os mesmo
	shift + f6 -> refactore , permite alterar o nome de uma variavel/metodo e muda também as referencias 
	
	
	ctrl + N -> FIND/procura Classes dentro do projeto (apenas clases) 
	ctrl + shift + N -> FIND/procura por todos os arquivos no projeto (apenas clases) 
	ctrl + shift + a -> Procura por ações do intelliJ (muito util)
		posso achar o "find usages" dentro dele
		
	Resumido
	ALT + 1 			- Open project tab 
	ALT + INSERT 		- Generate code/Create files
	CTRL + Z 			- Undo
	CTRL + ESPAÇO 		- Auto complete
	CTRL + D 			- Duplicate current line
	CTRL + SHIFT + F10 	- Create configuration and Run
	SHIFT + F10 		- Run
	CTRL + ALT + V 		- Pate
	CTRL + ALT + back 	- Navigate back
	CTRL + X 			- Cut
	CTRL + ALT + Foward - Navigate Foward
	CTRL + ALT + F 		- Extract Field
	CTRL + ALT + L 		- Reformat code 
	CTRL + ALT + O 		- Optimize imports 
	CTRL + B 			- Go to declaration
	CTRL + F4 			- Close active editor tab 
	ALT + F7 			- Find usages
	CTRL + ALT + M 		- Extract Method
	SHIFT + F6 			- Rename
	CTRL + N 			- Find class
	CTRL + SHIFT + N 	- Find files
	CTRL + SHIFT + A 	- Find actions
